Given a function f: bool -> bool -> bool, the function (binary_boolean_operator f), let's call it F, has type t -> t -> t and its semantics is as follows: Given A₁:t and A₂:t we have

F(A₁,A₂) = {x∊ℝ₊|f(i(x,A₁),i(x,A₂))}

where i(x,A₁) = true iff x∊A₁ ; i(x,A₂) = true iff x∊A₂

In particular, all the usual binary operators ⋂ (intersection), ⋃ (union), \ (difference) and Δ (symmetric difference) are obtained this way.
